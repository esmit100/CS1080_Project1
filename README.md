# Avalanche Awareness & Rescue (Multi-Page Website)

## Purpose
A 3-page informational website about avalanche awareness and rescue basics.

## Pages
- `index.html` – Home + quick checklist
- `learn.html` – Basics, planning template, links, embedded safety video (multimedia)
- `rescue.html` – Rescue workflow + gear table

## Key Features
- Semantic HTML structure
- 3 linked pages
- Image (local PNG & JPEG)
- Table (on Rescue page)
- Multimedia element (local video)
- Flexbox layout + responsive media query (cross over from Hackathon project)
- CSS styling for layout, typography, lists, and tables
- Optional JavaScript: tip toggle + quiz
